# Bill Gates Spend Clone

Bill Gates'in hayali $100,000,000,000 bakiyesini ürünler satın alarak harcadığınız basit bir klon uygulama.

## Özellikler
- Başlangıç bakiyesi: **$100,000,000,000**
- Ürün satın alma (Buy) ile bakiye azalır
- Ürün satma (Sell) ile bakiye artar
- Bir üründen hiç alınmadıysa **Sell** butonu pasif olur
- Ürün fiyatı bakiyeden büyükse **Buy** butonu pasif olur
- Satın alınan ürünler en altta **adet** ve **toplam tutar** ile listelenir (Receipt)

## Kullanılan Teknolojiler
- React
- TypeScript
- Vite
